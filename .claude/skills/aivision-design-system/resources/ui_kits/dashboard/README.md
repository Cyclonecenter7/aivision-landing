# Dashboard UI Kit

Вторая поверхность AIVISION — BI / дашборды / внутренние инструменты.

## Точка входа
`index.html` — полный демо-экран со всеми атомами. Скопировать и урезать под задачу.

## Подключение
```html
<link rel="stylesheet" href="../../colors_and_type.css" />  <!-- бренд-токены, всегда первым -->
<link rel="stylesheet" href="tokens.css" />                 <!-- dashboard-токены -->
<script type="text/babel" src="Primitives.jsx"></script>     <!-- все атомы в window.DB.* -->
```

## Атомы `window.DB.*`

| Атом | Когда использовать |
|---|---|
| `DB.Ticker` | LIVE-тикер — чёрная полоса в самом верху |
| `DB.KpiStrip` | Якорная полоса: 1 большая цифра + 3 вторичных |
| `DB.Kpi` | Отдельная KPI-карточка (размеры: compact / default / hero) |
| `DB.Sparkline` | Line chart для карточки или заголовка |
| `DB.SparkMini` | Tiny sparkline для ячейки таблицы |
| `DB.Bar` | Прогресс-бар (pos / neg / neu цвет) |
| `DB.Table` | Data table с кастомными рендерами колонок |
| `DB.Delta` | Дельта-ячейка (зелёный/красный +/−) |
| `DB.Funnel` | Воронка конверсии с меткой УЗЕЛ |
| `DB.PipelineColumn` + `DB.StageCard` | Kanban-колонка сделок |
| `DB.LineChart` | SVG line chart (факт + план-пунктир) |
| `DB.Callout` | Чёрная полоса с диагностической фразой AIVISION-тоном |
| `DB.DiagColumn` | Чёрная правая колонка «Что происходит · 3 узла» |
| `DB.FilterBar` | Горизонтальный ряд chip-фильтров |
| `DB.PeriodPicker` | Выбор периода (Неделя / Месяц / Квартал / Год) |
| `DB.Sidebar` | Левый сайдбар с группами ссылок |
| `DB.EditorialHead` | Editorial-заголовок-гипотеза («Октябрь. Узкое место — квалификация.») |

## Ключевые правила

- **3 data-цвета и только они:** `#16A34A` (рост), `#E5484D` (падение), `#3F6EE8` (объём/нейтральное).
- **Tabular-nums везде** где цифры. Без исключений.
- **Chamfer меньше, чем в маркетинге:** 4 / 6 / 8 / 10 / 12 / 16 / 20px.
- **PII-мок:** имена клиентов — класс `.db-pii` (blur 4px).
- **Графики — сырой SVG** через `DB.LineChart`. Recharts — только если нужна интерактивность.
- **Якорная цифра** — одна на экран, размер 72–96px, weight 800.
- **Диагностическая колонка** (`DB.DiagColumn`) — главный «дерзкий» элемент бренда на дашборде. Чёрный правый столбец в светлом интерфейсе.
